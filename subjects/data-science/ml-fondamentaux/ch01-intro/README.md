# Chapitre 1 — Introduction au ML

## Définition

Le **machine learning** est une branche de l'intelligence artificielle où des modèles apprennent à partir de données plutôt que d'être entièrement programmés à la main.

## Types d'apprentissage

- **Supervisé** : données étiquetées (X, y).
- **Non supervisé** : structure cachée sans labels.
- **Par renforcement** : récompenses et politique.

## Remarque

La **généralisation** sur données non vues est le critère principal de qualité, pas seulement l'erreur sur l'ensemble d'entraînement.
